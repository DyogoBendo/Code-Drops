# Funções Presentes

## formatMoney
Converte um valor inserido para BRL Real

## formatCPF
Transforma uma cadeia de números no formato convencionado para o CPF

# Instalação

```
npm i Code_Drops_22
```

```
const {formatMoney, formatCPF} = required('Code_Drops_22')
```

# Criando conta no NPM

```
npm adduser
npm whoami
```

# Adicionando no npm
```
npm init -y
```
